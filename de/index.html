Get started

1. Install and set up

For AMD modules, add these script tags in the head of your web page: 

                
                <link rel="stylesheet" href="https://js.arcgis.com/4.28/esri/themes/light/main.css">
                <script src="https://js.arcgis.com/4.28/"></script>

Then add this script to the last of your HTML page: 

                
                <script src="http://infopmo2030-001-site8.atempurl.com/esriGTLibrary/EsriFunctions.js"></script>

2. Add HTML

Define an HTML div element to display the map. 

                
                <div id="displayMap">
                     <div id="searchDiv">
                     </div>
                </div>

Then add this CSS to the CSS file of your web page: 

                
                  html,
                  body,
                  #displayMap {
                       padding: 0;
                       margin: 0;
                       height: 100%;
                       width: 100%;
                       -webkit-print-color-adjust: exact !important;
                       print-color-adjust: exact !important;
                  }
                  
                  .searchStyle {
                       position: absolute;
                       left: 3.2rem;
                       top: 9px;
                  }
                  
                  .esri-ui-top-left {
                       align-items: baseline !important;
                  }

Initialize the map

Call the initializeMap() function to display the map:

This function takes a parameters and it's about: 

                
                  initializeMap(arrayOfDisplayedGraphics, withSketch = false, HTMLElementsArray=[], AddModalId="", HasLabel=false)
                  // arrayOfDisplayedGraphics:  The array of the locations to display on the map (Array of objects),
                  // withSketch : Whether to have sketch widget or not(Boolean),
                  // HTMLElementsArray: All the html components to add in the map(Array of HTML elements[HTMLElement,Boolean])
                  // HTMLElementsArray[(HTMLElement), button will replace the search elemnt or not (Boolean)]
                  // HTMLElementsArray will send it to the addWidgets function that called inside the initializeMap function.
                  
                  // AddModalId: Uique ID for the html modal to perform the action of add  new location (String),
                  // HasLabel: Whether to display label for each location or not(Boolean)

Zoom To Coordinates

Call the ZoomTOCoordinates() function to zoom to specific location:

This function takes a parameters and it's about: 

                
                  ZoomTOCoordinates(id)
                  // id:  The id of the location that called (ProjectLocationId of ProjectData in the above object).

Remove Last Figure

Call the RemoveLastFigure() function:

This function don't takes any parameters: 

                
                  RemoveLastFigure()
                  // Use this function to remove last graphics if the user delete the graphics from the map.

Convert XY To Latitude, Longitude

Call the ConvertXY_To_LATLONG() function to convert from PCS to GCS:

This function takes a parameters and it's about: 

                
                  ConvertXY_To_LATLONG(graphic)
                  // graphic:  The graphic that displayed on the map in Projected Coordinate System.

Get Location Points Details

Call the GetLocationPointsDetails() function:

This function don't takes any parameters: 

                
                  GetLocationPointsDetails()
                  // This function return the geometry data of the graphic.

Step Through Graphics

Call the stepThroughGraphics() function to step step through graphics on the map:

This function takes a parameters and it's about: 

                
                  stepThroughGraphics(view, gra_MainGL, index, IsActive)
                  // view: The view of the map,
                  // gra_MainGL: The graphics layer that displayed the graphics on the map,
                  // index: The index of the graphic in the graphics layer,
                  // IsActive: A flag to confirm if you need to show the modal to the current graphic or not
